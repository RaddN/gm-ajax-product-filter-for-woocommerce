A WooCommerce plugin to filter products by attributes, categories, and tags using AJAX for seamless user experience.

## Description 

GM AJAX Product Filter for WooCommerce is a powerful plugin that enables WooCommerce stores to provide product filtering based on various parameters such as categories, tags, and attributes without requiring page reloads. This AJAX-powered filter enhances user experience, allowing for a smoother and more dynamic interaction.

## Features 

1. **Dynamic Filtering Options**
   - **By Query String**: Apply filters using query strings (e.g., `?filters`).
   - **With Permalinks**: Integrates with custom permalink structures (e.g., `canada/toronto/feb-2024`).
   - **AJAX-Powered**: Seamless, reload-free filtering for an improved shopping experience.

2. **Customizable Filter Management**
   - Manage filter options easily.
   - Filter products by **category**, **tag**, or **attribute**.

## Installation

1. Download the plugin from [Plugin URI](https://plugincy.com/).
2. Upload the `gm-ajax-product-filter` folder to the `/wp-content/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
4. Configure filter settings in the WooCommerce settings panel under "Product Filter."

## Usage

- Configure the plugin settings to define the available filter options by categories, tags, and attributes.
- Customers will be able to use the AJAX-based filtering without reloading the page, enhancing their shopping experience.
